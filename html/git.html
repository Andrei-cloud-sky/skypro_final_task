Git

    Команды для работы с репозиторием

    • git init — инициализирует пустой репозиторий.
    • git clone — создаёт локальную копию удалённого репозитория.
    • git remote add origin — связывает локальный и удалённый репозитории.

    Команды для работы с изменениями

    • git commit — фиксирует изменения в локальной истории репозитория.
    • git push — передаёт сделанные локально изменения в связанный удалённый репозиторий.

    Команды для работы с ветками

    • git branch — показывает список локальных веток и отмечает текущую звёздочкой.
    • git checkout — переключается между ветками или восстанавливает файлы в рабочем каталоге до состояния из определённого коммита или индекса.
    • git merge — сливает изменения из одной ветки в другую.
    • git branch -d — удаляет локальную ветку.
